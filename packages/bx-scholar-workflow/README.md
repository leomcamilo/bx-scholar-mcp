# BX-Scholar Workflow

Academic research workflow — prompts, skills (resources), and orchestrators for the full research lifecycle.

Part of the [bx-scholar-mcp](https://github.com/leomcamilo/bx-scholar-mcp) monorepo. Depends on [bx-scholar-core](../bx-scholar-core/).

## What's included

- **8 prompts** — research pipeline, journal calibrator, citation verification, literature search, R&R, qualitative analysis, theory development, meta-analysis
- **21 skill resources** — detailed protocols for every research phase
- **Orchestrators** — composite tools that chain core tools into workflows (PRISMA search, snowballing, journal DNA)

## Install

```bash
cd packages/bx-scholar-workflow
uv run bx-scholar-workflow
```

See the [monorepo README](../../README.md) for full documentation.
